# GM-Capstone
Capstone project for fouth year engineering project made in 2019. 

## **Introduction**

> The task given by GM was to redesign the current system that is in place that unloads the tires when near the end of the assembly line. The main issue with the current design is when the tires are unloaded by being placed on the currently installed system, the tires would apply a force horizontally on the conveyor belt resulting in the conveyor belt getting damaged.
> 
> 
> 

## **Objective**

> The goal of this project is to design a method to load the tires(push out the tires) before the vehicle reaches the conveyor belt. Another goal is to minimize the amount of parts that GM would have to replace during the modification of their current system.
>
